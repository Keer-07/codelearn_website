# codelearn_website

A modern and responsive educational website developed during my internship at **Yuva Intern** using **HTML, CSS, and JavaScript**.  
The website is designed to help beginners start their coding journey through interactive course sections, FAQs, responsive layouts, and user-friendly design.

---

## ✨ Features

- 🌟 Modern and clean UI design
- 📱 Fully responsive website
- 🎯 Smooth scrolling navigation
- 📚 Course catalog section
- 🚀 Interactive hero section
- ❓ FAQ accordion functionality
- 📩 Contact form with validation
- 🎨 Hover animations and transitions
- 🖥 Beginner-friendly learning platform layout

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Flexbox
- Responsive Web Design
- DOM Manipulation
